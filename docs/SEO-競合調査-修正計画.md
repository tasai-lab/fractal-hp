## 4. 現行フラクタルHPのSEO現状評価

### 優位点（競合を上回る部分）

**[S級] 構造化データの実装**
LocalBusiness + MedicalBusiness + Service + WebSite + FAQPage + JobPosting + Breadcrumb を全実装。競合で同レベルは0サイト。Google検索でのリッチリザルト（星評価・FAQ展開）の基盤が整っている。

**[S級] エリア別ランディングページ**
`/areas/[slug]/` の動的生成ページが存在。船橋市・八千代市・習志野市等の「エリア名 + 訪問看護」ロングテールキーワードで上位表示を狙える設計。競合で同様の施策を打っているサイトは0件。

**[A級] B2Bページ（ケアマネ・医療機関向け）**
`/for-care-managers/` と `/for-medical-institutions/` ページが存在。リニエのみが依頼書DLという形で対応しているが、フラクタルは専用ページとして分離。

**[A級] メタデータの品質**
タイトル・メタ説明・OGP・Twitter Card・Canonical URLが全て適切に設定。メタ説明に電話番号まで記載している（CTR向上施策）。

### 課題点（競合に劣る・未実装の部分）

**[P1: 最重要] ポータルサイト未掲載の可能性**
LIFULL介護（5位・32件掲載）とハートページナビ（8位・70件掲載）はそれぞれドメイン権威が高く、フラクタルが掲載されていれば2ページの流入源を追加獲得できる。これらへの掲載確認・申請が最も費用対効果の高い施策。

**[P1: 最重要] Googleビジネスプロフィールの最適化**
「船橋 訪問看護」でのローカル検索3パック（マップ表示）はマップ以外の検索結果より上に表示される。あおい訪問看護のように電話番号・口コミが充実していると、オーガニック検索より上位に表示される可能性がある。

**[P2: 重要] コンテンツ更新頻度**
リニエ（事業所ブログ）・あおい（頻繁なH2ページ更新）のように、定期的なコンテンツ更新がGoogleのクロール頻度と鮮度評価に直結する。更新情報ページ（`/updates/`）はあるが、独立したブログ・コラムコンテンツが未実装。

**[P2: 重要] スタッフ充実度の可視化**
くれよん（17名をH3で個別掲載）が2位に位置しているのは、スタッフ数とスタッフ情報の網羅性が評価されている可能性が高い。現行HPにスタッフ一覧はあるが、各スタッフのプロフィールページ（個別URL）は存在しない。

**[P3: 中優先] 料金ページの独立化**
きゃろっとがH2で「ご利用料金」を明示しているように、料金情報は検索ユーザーの主要な知りたいことの一つ。現行HPに独立した料金ページ（`/pricing/`）が未存在。

**[P3: 中優先] タイトルへの専門分野追記**
あおいのタイトル `【精神科・リハビリ・看取り対応】` のように、専門性を【】でタイトルに追加することでCTR向上が期待できる。現行タイトルの `【24時間対応】` は差別化ポイントとして有効だが、専門分野も追加する余地あり。

**[P3: 中優先] ふなぽーとへの掲載申請**
1位のふなぽーと（船橋市公式）は市内の訪問看護ステーション一覧を掲載。ここにフラクタルが掲載されれば、行政サイトからの被リンク（権威あるバックリンク）を獲得できる。

**現状**:
```

```
船橋市の訪問看護なら｜フラクタル訪問看護【精神科・リハビリ・看取り対応】24時間365日
