## 統合された改善提案

### 1. タイトルタグ最適化

**現状の問題点（両AIが指摘）:**
- 現在39文字で長すぎる（推奨28-32文字）
- 【】形式は有効だが、長すぎると省略される

**最終改善案:**

| ページ | 現在 | 改善案 |
|--------|------|--------|
| トップ | フラクタル訪問看護 船橋 \| 船橋市・八千代市・習志野市の訪問看護ステーション (39文字) | 船橋市の訪問看護｜フラクタル訪問看護【24時間対応】(26文字) |
| 採用 | 船橋市・八千代市・習志野市の訪問看護求人 \| フラクタル訪問看護 | 【船橋市】訪問看護師求人｜フラクタル訪問看護 (23文字) |

**各地域別ページ用タイトル:**
- `/areas/funabashi`: 船橋市の訪問看護｜フラクタル訪問看護 船橋
- `/areas/yachiyo`: 八千代市の訪問看護｜フラクタル訪問看護 船橋
- `/areas/narashino`: 習志野市の訪問看護｜フラクタル訪問看護 船橋

### 2. 地域別ページ戦略

**両AIの共通指摘:**
- 既存の地域ページは「採用」向けで、「利用者」向けページがない
- サービス利用者向けの地域別ページを新設すべき
- コンテンツの独自性が重要（テンプレ複製は避ける）

**最終方針:**

```
新設ページ構造:
/areas/
├── funabashi/        → 船橋市の訪問看護（利用者向け）
├── yachiyo/          → 八千代市の訪問看護（利用者向け）
├── narashino/        → 習志野市の訪問看護（利用者向け）
└── chiba-hanamigawa/ → 千葉市花見川区の訪問看護（利用者向け）

既存（採用向け）:
/recruit/areas/[slug] → そのまま維持
```

**各地域ページに含める独自コンテンツ:**
1. 地域の高齢化率・人口統計
2. 訪問可能な町名リスト
3. 連携医療機関（可能であれば）
4. 地域担当スタッフの紹介
5. 地域特化FAQ（「船橋市での訪問看護の料金は？」など）
6. 対応可能な医療処置の詳細
7. 地域からのアクセス情報
