# 採用ページ / フラクタルを知る ページ再設計（温かく洗練されたトーン）

作成日: 2026-01-31

## 概要

採用ページと「フラクタルを知る」ページを、温かく人間味のある質感と柔らかな色調で再設計する。情報量は維持しつつ、編集的なレイアウトで読みやすさ・信頼感・CVのしやすさを両立する。画像は自社/提供素材のみを使い、圧縮・最適化して体感速度を改善する。

## 目的

- 採用ページのCV向上を軸に、会社理解・共感を高める
- 「AIっぽさ」を感じさせない、温度感のあるデザインへ刷新
- 画像の最適化で表示速度と体験品質を改善

## 非目的

- 文面の大幅な刷新（必要最低限の調整のみ）
- 新規素材の外部調達（フリー/有料素材は使わない）

## 制約・前提

- 自社/提供素材のみ使用
- Google Fontsの追加はOK
- セクション構成は大きく再構成可
- 画像圧縮とフォーマット最適化を実施

## デザインアプローチ（推奨）

**温かい“編集的レイアウト”**
- 余白・長文の読みやすさ・写真の大きな扱いで信頼感を担保
- 小さなカード群ではなく、見開き風・コラム風のまとまりを作る
- CV導線は押しつけず、暖色系のカードで自然に視線誘導

## ビジュアルシステム

### カラー
- ベース: アイボリー / サンド（紙質感）
- アクセント: 柔らかいグリーン / くすみベージュ
- CTA: 暖色（淡いテラコッタ系）

例: 
- Background: #F7F2EA / #EFE7DD
- Primary: #2F5B4A
- Accent: #C78A67
- Muted: #7B6F63

### タイポ
- 見出し: Zen Old Mincho（明朝で品のある印象）
- 本文: M PLUS 1p（柔らかい丸ゴシック）
- H1〜H3の階層を明確化、本文行間を広めに

### 形状・質感
- 角はやや丸め（8px〜16px）
- 背景に薄い紙テクスチャ / ぼかしグラデーション
- 写真の上に薄いレイヤー（暖色）を重ね、温度感を統一

## レイアウト設計

### 採用ページ
1. ヒーロー（人物写真 + 短い理念コピー + CTA）
2. 働きやすさ・制度（編集的グリッドで要点を提示）
3. 現場の空気（写真ストーリーの帯）
4. 職種概要（カード化、長文は分割）
5. FAQ（柔らかいアコーディオン）
6. 応募導線（カード型CTA）

### フラクタルを知る
1. イントロ（写真 + 短い導入）
2. 理念（見開き風の余白設計）
3. DAPAE（横長タイムライン・物語形式）
4. 行動指針（コラムカード群）
5. ロゴの意味（写真×テキスト対比）
6. 結語（短い締めの言葉）

## コンポーネント方針

- 編集的見出しブロック（小見出し＋短文）
- 温かい統計カード（数字 + 説明）
- 写真ストーリー帯（横長ビジュアル）
- コラムカード（短文の読み物化）
- CTAカード（説明付き・柔らかなボタン）

## モーション

- ページロード時: フェード + スタッガー
- スクロール時: ゆるい浮遊（低速）
- カード: 軽いシャドウ変化

※過剰な動きは避ける

## 画像最適化方針

- 既存素材を用途別にリサイズ
- WebP/AVIFを作成し、`next/image`で`priority/sizes/quality`を適切に指定
- オリジナルは保持し、表示用は最適化フォルダに配置

推奨サイズ目安:
- ヒーロー: 2400px
- セクション: 1600px
- カード: 1200px
- サムネイル: 800px

## パフォーマンス・品質確認

- LighthouseでLCP/CLS/FIDを確認
- 画像のロード順（priority/placeholder）を調整
- モバイルでのテキスト可読性を確認
