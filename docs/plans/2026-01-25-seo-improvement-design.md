# SEO改善設計書

## 概要

「船橋 訪問看護」「船橋 看護師 求人」「八千代市」「習志野市」「花見川区」などのキーワードで検索上位表示を目指すSEO改善。

## 優先順位

1. 技術的SEO強化（構造化データ追加）
2. 既存ページの最適化（求人ページ優先）
3. コンテンツ拡充（低優先）
4. 地域別ページ作成（低優先・今回スコープ外）

## 1. 構造化データの追加

### 1.1 FAQスキーマ

#### サービス利用者向けFAQ（トップページ）

| 質問 | 回答概要 |
|------|----------|
| 訪問看護とは何ですか？ | 看護師が自宅を訪問し、療養生活を支援するサービス |
| どのような方が利用できますか？ | 病気や障がいがあり、主治医が訪問看護を必要と認めた方 |
| 利用料金はいくらですか？ | 医療保険・介護保険適用で1〜3割負担 |
| 利用開始までの流れは？ | 主治医への相談→指示書発行→契約→サービス開始 |
| どのエリアに対応していますか？ | 船橋市、八千代市、習志野市、千葉市花見川区、千葉市美浜区 |

#### 求職者向けFAQ（求人ページ）

| 質問 | 回答概要 |
|------|----------|
| 未経験でも応募できますか？ | 訪問看護未経験でも応募可能、先輩同行で丁寧に指導 |

| 入社祝い金の条件は？ | 本HPからの応募限定で最大30万円 |
| 訪問エリアはどこですか？ | 船橋市、八千代市、習志野市、千葉市花見川区 |

### 1.2 サービススキーマ

現在の`MedicalProcedure`を拡張し、各サービスに対応エリアを明記:

```json
{
  "@type": "MedicalProcedure",
  "name": "訪問看護",
  "description": "船橋市・八千代市・習志野市・花見川区でご自宅への看護ケア",
  "areaServed": ["船橋市", "八千代市", "習志野市", "千葉市花見川区"]
}
```

### 1.3 パンくずリスト

各ページに階層構造を追加:

- `/recruit`: ホーム > 採用情報
- `/about-fractal`: ホーム > フラクタルを知る
- `/flyers`: ホーム > チラシ

## 2. 求人ページの最適化

### 2.1 見出し構造の改善

#### 現在 → 改善後

| 要素 | 現在 | 改善後 |
|------|------|--------|
| H1 | 採用情報 | 船橋市・八千代市・習志野市の訪問看護師求人 |
| H2（募集） | 募集職種 | 船橋エリアの訪問看護師・セラピスト募集 |
| H2（エリア） | 訪問エリア | 訪問エリア：船橋市・八千代市・習志野市・花見川区 |
| H4（仕事） | 仕事内容 | 訪問看護師の仕事内容 |

### 2.2 リード文の追加

ページ冒頭（H1直下）に地域キーワードを含む導入文を配置:

```
船橋市、八千代市、習志野市、千葉市花見川区で訪問看護師・理学療法士・作業療法士・言語聴覚士を募集しています。
入社祝い金最大30万円、年間休日139日以上。未経験・ブランクのある方も歓迎です。
```

### 2.3 コンポーネント構造の改善

タブ切り替え等のインタラクティブ部分のみをクライアントコンポーネントに分離:

```
recruit/page.tsx (Server Component)
├── H1, H2, リード文（SEO重要テキスト）
├── RecruitTabs.tsx (Client Component) - タブ切り替えUI
└── FAQスキーマ
```

## 7. 検証方法

1. **Google Search Console**: インデックス状況、検索パフォーマンス
2. **リッチリザルトテスト**: 構造化データの検出確認
3. **PageSpeed Insights**: Core Web Vitals確認
4. **実際の検索**: ターゲットキーワードでの順位確認（デプロイ後2〜4週間）
